Эта Terraform конфигурация создает полную инфраструктуру в Yandex Cloud для вашего .NET микросервиса, включая:

VPC и сетевую инфраструктуру с NAT Gateway

Managed Kubernetes кластер с несколькими группами узлов

Managed PostgreSQL кластер с репликацией

Container Registry для образов Docker

Load Balancer с SSL сертификатами

DNS и мониторинг

Безопасность и IAM

Все настройки параметризованы и готовы к использованию в production.
