# План реализации: Лендинг платформы онлайн-обучения

## Обзор

Реализация простого лендинга на базе Piranha CMS без логики аутентификации. Проект использует ASP.NET Core 8.0 для серверной части с Razor Pages для рендеринга. Вся логика авторизации и личного кабинета вынесена в отдельный фронтенд-микросервис.

Микросервис располагается в структуре проекта по адресу: `apps/landing-cms/`

## Примечания

- Задачи, помеченные `*`, являются опциональными и могут быть пропущены для более быстрого MVP
- Каждая задача ссылается на конкретные требования для отслеживаемости
- Контрольные точки обеспечивают инкрементальную валидацию
- Property-тесты валидируют универсальные свойства корректности
- Unit-тесты валидируют конкретные примеры и граничные случаи
- Лендинг НЕ содержит логики аутентификации - только редиректы на /app
- Все формы авторизации, регистрации и личный кабинет находятся в отдельном фронтенд-микросервисе
